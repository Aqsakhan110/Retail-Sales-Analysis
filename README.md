# Retail Sales Dashboard (Excel)

Interactive Excel dashboard designed to analyze retail sales performance, monitor key business metrics, and support data-driven decision-making. The dashboard provides insights into product performance, regional sales, customer behavior, and payment preferences through interactive visualizations and filters.

## Key Features

- KPI Overview (Total Revenue, Total Profit, Total Orders, Profit Margin)
- Revenue by Product Analysis
- Monthly Sales Trend
- Top Region by Revenue
- Customer Type Analysis
- Payment Method Distribution
- Interactive Slicers for dynamic filtering
- Business Insights & Recommendations

## Dataset

The dashboard is built using a retail sales dataset containing transaction-level records, including product details, customer type, region, payment method, sales amount, profit, and order information.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Conditional Formatting
- Dashboard Design
